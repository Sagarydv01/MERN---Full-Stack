React JS:

* frontend JS library
* use to create single-page applications (SPA)
* lightweight
* extension= .jsx
* uses Virtual DOM



Library:

* collection of pre-written code
* perform specific tasks



Framework:

* complete structure
* provides rules, patterns, and tools



SPA:

* loads a single page and updates content dynamically without reloading the page.
* faster after the first load, as only necessary data is updated. -> lazy loader
* uses client-side routing (React Router) to switch views without reloading.
* Example: Gmail, FB, Netflix, YouTube Web App





MPA:

* each page is a separated HTML file, making navigation slower.
* slower performance, as the browser loads everything again on each page change.
* every new page request reloads the entire page from server
* uses server-side routing (URLs trigger full-page reloads)
* Example: Amazon, Wikipedia, News Websites, Banking Portals
